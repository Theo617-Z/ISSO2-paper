# IS-SO2
This repository contains all data and codes for IS-SO2 paper

## Data
It contains original data and preprocessing code and preprocessed data
- hko_23ap_ha_0516.rda: Raw source data used for the analysis.
- preprocess_data.R: prepares the analysis dataset by selecting variables, restricting years to 2008-2016, log-transforming ischemic stroke admissions, detrending, deseasonalizing, and standardizing numeric series.
- analysis_data.cdv: preprocessed analysis-ready dataset generated by `preprocess_data.R`.

## Codes
It contains the three minimal analysis scripts. These scripts read `Data/analysis_data.csv` and output numerical result tables needed before plotting.
- CCM_Smap.R: runs convergent cross mapping and S-map coefficient analysis for ischemic stroke against PM2.5, O3, SO2, temperature, and humidity across lag 0-3. Outputs `ccm_res.csv` and `smap_res.csv`.
- PCMCI_plus.py: runs PCMCI+ with ParCorr conditional independence testing for lagged causal structure. Outputs `pcmci_long_Overall_Year.csv`.
- GLM.R: runs lag-specific generalized linear models using the PCMCI-informed adjustment structure. Outputs `glm_res.csv`.

## Notes
- The scripts generate data tables only; they do not produce figures.
- Run `Data/preprocess_data.R` first
